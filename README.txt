The person names in personnames_table.csv are a compilation of the Wikipedia lists for
- female givennames (<http://de.wikipedia.org/w/index.php?title=Kategorie:Weiblicher_Vorname>)
- male givennames (<http://de.wikipedia.org/w/index.php?title=Kategorie:MÃ¤nnlicher_Vorname>)

The events in events_table.csv are a compilation of
- small hand made list for German
- Wordnet-extracted for English
For German there is a java program that generates a table from GermaNet.
